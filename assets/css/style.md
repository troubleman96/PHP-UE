# assets/css/style.css

## Purpose

`assets/css/style.css` controls the design of the app.

## What It Styles

- Page width and spacing
- Buttons
- Cards
- Forms
- Flash messages
- Tables
- Mobile layout

## Important CSS Classes

- `.page` for the main page width
- `.hero` for the landing page section
- `.card` for content boxes
- `.actions` and `.form-actions` for button layout
- `.flash` for success and error messages
- `.grid` for the two-column layout
- `.table-wrap` for the notes table

## In Simple Words

This file does not handle any PHP logic. It only makes the pages look clean and
organized.
